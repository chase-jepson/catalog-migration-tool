    - detectPOS: Given headers matching Dutchie defaults, returns { detected: "Dutchie" }
    - detectPOS: Given headers matching no POS (threshold: >=3 AND >40%), returns { detected: null }
    - detectPOS: Given multiple files with different POS matches, returns majority with disagreement: true
    - scorePOS: Given "Dutchie" and Dutchie headers, returns correct matched/total counts
    - applyPOSDefaults: Given "Dutchie", returns FieldMapping[] with Dutchie-specific sourceHeaders populated
    - applyPOSDefaults: Given "Other", returns empty mappings (no auto-map)
    - updateMapping: Given mappings and a fieldKey+sourceHeader, returns updated mappings with that field changed
    - getUnmappedRequired: Given mappings with unmapped required fields, returns list of unmapped required field keys

       - `detectPOS(files: ParsedFile[]): POSDetectionResult` -- iterate POS_SYSTEMS, score each against file headers. Threshold: matched >= 3 AND matched/total > 0.4. For multi-file: majority vote. Set disagreement: true if files vote differently.
