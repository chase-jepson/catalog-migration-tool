export interface ETLInput {
  inventoryFile: ParsedFile;          // required
  receiptsFile?: ParsedFile;          // optional

Implement each phase as a standalone exported function:

- `processReceipts(receiptRows, adjustmentRows, inventoryExternalPackageIds, receiptMappings, adjustmentMappings)`:
  1. Build fieldMap for receipts and adjustments
  2. Extract receipt subset columns via getVal
  3. sumByGroup receipts on [ProductSKU, ReceiveDate, ExternalPackageID, VendorName, OrderTitle] -> sum Quantity, TotalCost
  4. sumByGroup adjustments on ExternalPackageID -> sum Quantity, TotalCost
  5. Stack receipt totals + adjustment totals
  6. sumByGroup combined on ExternalPackageID -> final Quantity, TotalCost
  7. Left join: invoice descriptive rows + combined totals on ExternalPackageID
  8. Calculate UnitCost = round(TotalCost / Quantity, 2)
  9. Extract InvoiceId from OrderTitle
